##__title__ Exibir uma galeria de imagens com uma imagem maior em destaque. O usuário vai poder navegar entre as imagens clicando nos thumbnails abaixo ou utilizando as setas direcionais (Anterior e Próximo) Quando uma imagem em destaque, deve ser exibido a posição dela na lista de thumbnails e a legenda. O texto da legenda deve estar no atributo alt do thumbnail.

![Wireframe](wireframe.jpg "Logo Title Text 1") ###Desafio #2 O link para as imagens devem estar em um arquivo [.json](https://www.youtube.com/watch?v=t4Y7jd4h-T8) e não no HTML da página. Você vai precisar pesquisar como realizar uma requisição Ajax utilizando somente javascript. Lembra que jQuery nesse momento da vida não serve. ###Desafio #1 Quando o usuário não estiver com o mouse em cima do mouse, após 10 segundos, as imagens devem começar a passar sozinhas.
